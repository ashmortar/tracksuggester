You should consider the PHP/Drupal track

PHP is a backend language used by most websites. It is a versitle language that is used to manage content for things like Wordpress, Drupal and Joomla but is used in many different applications and in many different organizations. You can learn more about the PHP course here and more about the Drupal course here
